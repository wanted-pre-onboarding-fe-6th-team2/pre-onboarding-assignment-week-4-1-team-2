# React-Vite-JS-Boilerplate

Vite에서 제공하는 (React + Vite + JavaScript) + 추가적인 프로젝트 초기 세팅을 기반으로 한 보일러 플레이트입니다.

## 의존성

보일러 플레이트에 포함 된 의존성은 다음과 같습니다.

### dependencies

- emotion
- react-router-dom

### devDependencies

- eslint
- prettier
- husky
- lint-staged
